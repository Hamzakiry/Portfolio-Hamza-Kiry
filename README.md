# Hamza Kiry – Developer Portfolio

Bienvenue sur mon portfolio développeur.  
Ce projet a été conçu pour présenter mon profil, mes projets, mes compétences ainsi que mes liens professionnels dans une interface moderne, responsive et visuellement engageante.

## Aperçu

Ce portfolio personnel met en avant :

- Une page d’accueil moderne avec animation visuelle
- Une section de présentation
- Une mise en avant de plusieurs projets
- Une section contact avec liens vers GitHub, LinkedIn et Gmail
- Un design responsive adapté aux différents écrans

## Technologies utilisées

Ce projet a été développé avec les technologies suivantes :

- **React.js**
- **JavaScript**
- **Tailwind CSS**
- **React Icons**
- Composants UI personnalisés
- Effets visuels interactifs

## Fonctionnalités

- Navigation responsive
- Hero section animée
- Présentation personnelle
- Galerie de projets
- Footer avec liens professionnels
- Interface moderne et épurée
